Tags sematicas em HTML 5

As tags semanticas são elementos HTML que descrevem claramente o sigificado do conteudo que elas envolvem. Elas ajudam a melhorar a acessibilidade , a SEO e a estrutura do documento. Aqui estao algumas das pricipais tags semanticas:

  • <header>: Define o cabeçalho em um documento ou seção.
  • <nav>: Define um conjunto de links de navegaçao
  • <main>: define o conteudo principal do documento
  • <section>: define uma seçao generica de um documento
  • <article>: Define um conteudo independente e auto-contido, como um artigo de blog ou noticia.
  • <aside>: define um conteudo mas separado do conteudo principal com uma barra lateral
  • <footer>: define o rodape de um documento ou seçao

Usar essas tags semanticas ajuda os motores de busca a entenderem melhor o conteudo do seu site e melhora a experiencia do usuario especialmente para aqueles que utilizam leitores de tela.
